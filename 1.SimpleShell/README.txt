Problem Statement: write a program to implement a shell, that performs three essential functions:	1. Interact with a human user by reading and parsing input from the keyboard, and writing output to the terminal.	2. Create and manage processes as specified by the commands typed by the user.	3. Arrange for input and output to go to the right places (files or other processes).	Also, your shell should have a ``built in'' command called about that prints a string including your name and student id.	Your shell should handle command pipelines and I/O redirection as well as simple commands. To accomplish this, you will need to use the dup() and pipe() system calls, as well as open(), close(), fork(), and exit().
Compile and Execute Instructions---------------------------------------------------------
Below are the files for ‘Project 1 Simple Shell’:
	main.c - source code
	myShell.h - function declarations
	Makefile - commands to compile, run, clean
Below are the lists of commands from 'Makefile' to compile and run the 'Project 1 Simple Shell':	1. make all - Command to compile the source code	2. make run - Command to run 'mysh' executable created after compilation	3. make clean - Command to clean residue i.e. 'mysh' executableBelow is the flow of the shell after running 'mysh':	1. Command prompt is printed on terminal where user enters the command	2. Enter the command	3. Press enter	4. User gets desired outputCommands supported:	Simple commands – about, cd, clr, dir, environ, echo, help	1. exit - Terminates shell	2. ls - Command with no arguments	3. ls -l - Command with arguments	4. ls -l > foo - Output redirection with or without arguments	5. sort < testfile - Input redirection with or without arguments	6. ls -l | more - Pipe with or without arguments**********************************************************Results----------------------------------------------------------shweta@ubuntu:~/Desktop/Project$ make clean && make all && make runrm -f *.o myshgcc -g -o mysh main.c./mysh************* PROJECT: SIMPLE SHELL ************** Enter cmd #1. ‘about’ built in commandEnter cmd # aboutStudent Name : Shweta Kharat Student Id   : 000----------------------------------------------------------
